ANKI 單字卡生成器

輸入單字 → AI 生成內容 → 直接推進 ANKI

首次使用？點這裡看完整設定步驟
【第一階段】桌機一次性設定（建立樣板）
  1. 桌機安裝 AnkiConnect：開啟 ANKI → 「工具」→「附加元件」→「取得附加元件」→ 輸入 2055492159 → 確定 → 重新啟動 ANKI
  2. 在 ANKI「工具」→「附加元件」→ 選 AnkiConnect → 「設定」，貼上下方設定，按確定後重啟 ANKI
  3. 桌面雙擊 anki單字卡.html 用 Chrome 開啟（必須從本機檔案開，不能用 Netlify 網址）
  4. 填入 Groq API Key → 輸入任一單字 → 生成卡片 → 點「✚ 加入 ANKI（桌機）」
  5. Anki 桌機同步 → 手機 AnkiMobile「同步集合」
AnkiConnect 設定（全選複製貼上）：
{
    "apiKey": null,
    "apiLogPath": null,
    "ignoreOriginList": [],
    "webBindAddress": "127.0.0.1",
    "webBindPort": 8765,
    "webCorsOriginList": [
        "http://localhost",
        "null"
    ]
}
【第二階段】日常手機使用流程
  1. 用 Safari 開啟 https://anki-vocab-adder.netlify.app（Chrome 或 Telegram 內建瀏覽器無法跳轉 AnkiMobile，只能用 Safari）
  2. 點「⚙ 設定」→ 填入 Groq API Key、設定牌組名稱（必須和 Anki 裡已有的牌組名稱一致）
  3. 輸入單字 → 生成卡片 → 點「📱 加入 AnkiMobile（手機）」
  4. 自動跳轉 AnkiMobile → 確認儲存
建議：Safari 開啟後點「分享」→「加入主畫面」，下次像 App 一樣直接點開
【選配】改善 TTS 聲音品質
  1. iPhone 設定 → 輔助使用 → 語音內容 → 聲音 → English
  2. 選「Samantha」或「Ava」→ 點右邊下載 → 選「Enhanced（高品質）」
  3. 下載完成後 AnkiMobile 複習時聲音更清晰
